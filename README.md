# CuidaMed Pill

## Descripción del proyecto

Actualmente, la tecnología forma parte de la vida diaria, aunque muchas personas mayores tienen dificultades para gestionar tareas digitales, como el seguimiento de su medicación. Para ayudar en este proceso surge CuidaMed Pill, una aplicación móvil que funciona como un pastillero digital, permitiendo registrar medicamentos, controlar las tomas y recibir recordatorios personalizados.


La aplicación destaca por su accesibilidad, ya que ofrece una interfaz sencilla y adaptada para usuarios con poca experiencia tecnológica, además de permitir la supervisión remota por parte de tutores o cuidadores. Asimismo, se realizaron pruebas con usuarios para mejorar su usabilidad.


En conclusión, CuidaMed Pill proporciona una solución práctica y accesible que facilita el seguimiento de la medicación diaria y contribuye a reducir olvidos y aumentar la seguridad de los pacientes.



- Este proyecto ha sido desarrollado como Trabajo de Fin de Grado (TFG), incluyendo el análisis de necesidades, diseño de la interfaz, implementación de la aplicación y pruebas con usuarios.

---

## Funcionalidades

La aplicación CuidaMed Pill incluye las siguientes funcionalidades principales:

### Gestión de usuarios
- Registro y restricción de funcionalidades de usuarios asistidos
- Registro de usuarios estándar

#### Supervisión de usuarios asistidos
- Vinculación entre tutores y usuarios asistidos.
- Seguimiento del cumplimiento del tratamiento.
- Administración de las distintas opciones de forma remota por parte del tutor al asistido.

### Gestión de medicación
- Registro, edición y eliminación de medicamentos.
- Configuración de horarios, notificaciones, y personalización del fármaco.

### Sistema de recordatorios
- Notificaciones personalizables para la toma de medicamentos.
- Configuración de avisos.


### Interfaz accesible
- Diseño simplificado orientado a personas mayores.
- Navegación clara y estructurada.

---

## Arquitectura del sistema

El sistema ha sido diseñado siguiendo una arquitectura modular, separando claramente:

- Lógica de negocio
- Gestión de datos
- Interfaz de usuario
- Sistema de notificaciones

Esta estructura facilita el mantenimiento, la escalabilidad y futuras ampliaciones del proyecto.

## Tecnologías utilizadas

- Lenguaje de programación: Java
- Framework: Android Studio
- Base de datos: Firebase
- Herramientas de diseño: aplicaciones de diseño digitales
- Control de versiones: GitHub

## Pruebas

Se han realizado pruebas con usuarios potenciales para evaluar la usabilidad y funcionalidad de la aplicación.

Los aspectos evaluados incluyen:

- Facilidad de uso de la interfaz
- Comprensión del sistema
- Utilidad de los recordatorios
- Gestión de medicamentos
- Experiencia general de uso

Los resultados obtenidos han permitido identificar mejoras en accesibilidad y usabilidad, especialmente orientadas a personas mayores.
